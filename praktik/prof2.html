Функции в JavaScript

Функция, которая принимает массив чисел и возвращает количество четных элементов, расположенных выше главной диагонали:


function countEvenAboveDiagonal(matrix) {
    let count = 0; // Начинаем с нуля
    for (let i = 0; i < matrix.length; i++) {
        for (let j = i + 1; j < matrix[i].length; j++) {
            if (matrix[i][j] % 2 === 0) {
                count++;
            }
        }
    }
    return count;
}

Пример использования функции:


const matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

console.log(countEvenAboveDiagonal(matrix)); // Вывод: 1 (только число 6)
